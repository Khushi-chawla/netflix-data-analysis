Data Analysis Project exploring Netflix movies and TV shows using SQL, Python and Data Visualization. 
#  Netflix Content Analytics | SQL • Python • Power BI

## Project Overview

This project analyzes Netflix's movies and TV shows dataset to uncover trends in content distribution, ratings, release patterns, genres, and country-wise production. The analysis combines SQL for querying, Python for data cleaning and exploratory analysis, and Power BI for interactive dashboard creation.

## Objectives

- Clean and prepare the Netflix dataset
- Perform exploratory data analysis
- Write SQL queries to answer business questions
- Build interactive Power BI dashboards
- Generate meaningful business insights from the data

## Tools & Technologies

- SQL
- Python (Pandas, NumPy, Matplotlib)
- Power BI
- Microsoft Excel
  
## Project Workflow

1. Data Cleaning using Python
2. Exploratory Data Analysis (EDA)
3. SQL Query Analysis
4. Dashboard Development in Power BI
5. Business Insight Generation

## Dashboard Highlights

- Movies vs TV Shows
- Content by Country
- Content by Rating
- Release Year Trends
- Genre Distribution
- Content Added Over Time

## Key Insights

- Movies dominate Netflix's content library.
- The United States contributes the highest number of titles.
- TV-MA is the most common maturity rating.
- Netflix experienced a significant increase in content additions, peaking around 2021.
- Drama and International content are among the platform's most prominent genres.
